Algoritmo de Ford–Fulkerson

El objetivo es descubrir la mayor cantidad total de flujo que puede enviarse desde la fuente s hasta el sumidero t. Para lograrlo, el algoritmo usa uno o varios caminos disponibles, envía por cada uno la cantidad permitida y continúa hasta que ya no exista una ruta completa con capacidad.

Los valores se leen como flujo/capacidad

Criterio para elegir el camino

Ford–Fulkerson no exige escoger el camino con mayor capacidad ni el más corto. Puede usarse cualquier camino completo desde s hasta t cuyas aristas tengan capacidad residual positiva. En esta demostración se sigue un orden fijo para que el procedimiento sea fácil de observar.
