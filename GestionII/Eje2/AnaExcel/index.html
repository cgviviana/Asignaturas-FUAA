Análisis de datos en Excel desde una base de datos
Temas
¿Qué es el análisis de datos en Excel?

Analizar datos en Excel es revisar una base de datos para encontrar información útil. No se trata solo de mirar números, sino de organizarlos, compararlos e interpretarlos.

Idea clave

La base de datos contiene datos. El análisis convierte esos datos en información.

  • Encontrar patrones
  • Comparar grupos o periodos
  • Resumir información
  • Tomar decisiones
Ejemplo

Si una base registra llegadas tarde, Excel permite saber qué curso presenta más retrasos, qué días ocurre más y cuántos casos existen.

Limpieza de la base de datos

Antes de analizar, la base debe estar limpia. Si la base tiene errores, los resultados también tendrán errores.

Qué revisar

  • Filas vacías
  • Datos repetidos
  • Nombres mal escritos
  • Números guardados como texto
  • Fechas mal registradas

Herramientas de Excel

  • Ordenar
  • Filtro
  • Quitar duplicados
  • Buscar y reemplazar
  • Formato de celda
Ejemplo

Si en una fila aparece “10A” y en otra “10 A”, Excel podría contarlos como grupos distintos.

  • No dejar columnas sin nombre.
  • No unir celdas dentro de la base.
  • Una fila = un registro.
  • Una columna = un tipo de dato.
  • Todos los datos del mismo tipo deben tener el mismo formato.
Organización de datos

Una base ordenada permite trabajar mejor en filtros, fórmulas, gráficos y tablas dinámicas.

Importante

No se debe mezclar en una sola columna información diferente como “Juan - 10A - tarde”. Eso debe ir separado.

  • Facilita los filtros
  • Evita errores en fórmulas
  • Permite comparar
  • Hace posible la tabla dinámica
Comparativos

Comparar datos es mirar diferencias o comportamientos entre grupos, fechas, categorías o cantidades.

Comparativos frecuentes

  • Curso vs cantidad de casos
  • Día vs frecuencia
  • Persona vs rendimiento

Preguntas guía

  • ¿Cuál tiene más?
  • ¿Cuál tiene menos?
  • ¿En qué día ocurre más?
  • ¿Qué grupo presenta mayor frecuencia?
Ejemplo

Comparar cursos permite identificar cuál presenta mayor número de llegadas tarde.

Cálculos básicos en Excel

Después de limpiar y organizar, se aplican cálculos sencillos para resumir información.

CONTAR.SI = contar cuántas veces aparece una condición
SUMAR.SI = sumar según una condición
PROMEDIO = calcular media
MAX = valor mayor
MIN = valor menor
Ejemplos

CONTAR.SI puede servir para saber cuántos estudiantes llegaron tarde.

MAX puede mostrar el mayor número de minutos de retraso.

El cálculo por sí solo no basta. Se debe explicar qué significa.

Ejemplo

Si el curso 10A tiene 15 tardanzas y 10B tiene 6, entonces 10A presenta mayor frecuencia de retraso.

El gráfico no reemplaza el análisis. Solo ayuda a visualizarlo.

Tabla dinámica

La tabla dinámica es una herramienta de Excel que resume grandes cantidades de datos de forma rápida.

Permite

  • Contar
  • Sumar
  • Agrupar
  • Filtrar
  • Comparar

Estructura

  • Filas
  • Columnas
  • Valores
  • Filtros
Ejemplo

Filas: Curso

Columnas: Día

Valores: Cantidad de tardanzas

Resultado

Con una sola tabla se pueden ver patrones por curso y por día sin hacer conteos manuales.

  1. Seleccionar la base de datos
  2. Ir a Insertar
  3. Elegir Tabla dinámica
  4. Ubicar campos en filas, columnas y valores
  5. Interpretar el resultado
