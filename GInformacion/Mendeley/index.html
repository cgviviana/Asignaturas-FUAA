Acceso e instalación

Cuenta y sincronización

  • Regístrate en mendeley.com e inicia sesión en web y escritorio.
  • Activa Sync para mantener todo alineado entre dispositivos.

Desktop (Reference Manager)

  • Instala el gestor de escritorio para edición completa de metadatos y lectura de PDFs.
  • Opcional: Plugin de citas para Word/LibreOffice (Tools → Install ...).

Web Importer

  • Instala la extensión del navegador (Save to Mendeley).
  • Desde bases de datos/Google Scholar: guarda con un clic y revisa metadatos.

Tips

  • Desactiva bloqueadores si el importador no abre.
  • Verifica DOI/ISBN tras importar para evitar errores.

Interfaz principal

Vista clásica en tres paneles: a) Biblioteca/Grupos, b) Lista de documentos, c) Detalles/Notas. Barra superior con acciones.

a) Biblioteca

  • Carpetas por proyecto/asignatura.
  • Etiquetas, filtros por autor/año.

b) Lista

  • Estado leído ⭐, favoritos.
  • Ordenar por año, revista, autor.

c) Detalles

  • Editar metadatos y DOI.
  • Vista previa del PDF y notas.

Añadir y organizar referencias

1) Añadir Carpetas

  • En My Library crea Collections (proyecto/curso).

2) Añadir Referencias

  • Manual: Add Entry Manually.
  • Archivos: Add Files (PDF/ris/bib).
  • Web: con Web Importer desde bases de datos/Scholar.

3) Exportar Referencias

  • File → Export en RIS/BibTeX/XML.
  • Útil para migrar o usar en otras apps.

4) Trabajar con Documentos

  • Resalta y añade notas en el lector PDF.
  • Busca texto en PDF y metadatos.

5) Eliminar Duplicados

  • Tools → Check for Duplicates y Merge.

6) Organizar Ficheros

  • Options → File Organizer para renombrar por autor/año.

Uso en la Web (Mendeley Web)

Acceso

  • Entra a mendeley.com → Sign in.
  • Tu biblioteca se sincroniza con el escritorio.

Paneles Web

  • Dashboard: actividad de contactos y grupos.
  • My Library: gestionar referencias en línea.
  • Groups: colaborar y compartir PDFs.
  • Papers/People: descubrir tendencias y autores.

Guardar desde la Web

  • Con Web Importer guarda artículos desde bases de datos.
  • Revisa metadatos, corrige y Save.

Buen uso

  • Verifica campos clave (autor, año, revista, DOI).
  • Etiqueta por tema/proyecto para búsquedas rápidas.

Citar: Word y Google Docs

Word / LibreOffice

  • Instala el plugin: Tools → Install ...
  • Insert Citation → busca autor/título.
  • Insert Bibliography y cambia estilo en View → Citation Style.

Google Docs (método práctico)

  • En Mendeley: Edit → Copy Citation y pega en Docs.
  • Para la bibliografía: File → Export → RIS/BibTeX y usa un complemento (p. ej. Cite This For Me/BibGuru) para formatear.
  • Consejo: define APA 7 desde el inicio.

Precisión

  • Tras pegar, revisa mayúsculas en títulos y nombres propios.
  • Asegura DOI/URL y fechas de acceso según el estilo.

Atajo rápido

  • Arrastrar y soltar desde Mendeley inserta texto de cita (no dinámico).
  • Usa Refresh en Word para actualizar todas las citas.

Compartir y colaborar

Grupos

  • My Library → Groups → Create Group.
  • Invita por correo y define permisos.
  • Adjunta PDFs y referencias al grupo.

Buenas prácticas

  • Carpetas por equipo y por objetivo.
  • Nombrado consistente de archivos.

Descubrir

  • Explora autores y papers desde el panel web.
  • Sigue temas para alertas de novedades.

Backups

  • Exporta RIS/BibTeX periódicamente.
  • Respalda la carpeta de PDFs organizada.
