Economic Activity and CIIU Code

An economic activity is any work that generates goods or services. They are classified as:

  • Primary: Agriculture, fishing, mining.
  • Secondary: Industry, manufacturing.
  • Tertiary: Services like health, tourism, education.

The CIIU code identifies the specific activity. For example, a coffee shop uses code 5611: food served to the table.

Types of Companies
