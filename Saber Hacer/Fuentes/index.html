Fuentes de Información

Las fuentes de información son recursos utilizados para obtener datos y conocimientos sobre un tema específico. Pueden incluir documentos, bases de datos, libros, artículos académicos, sitios web, entrevistas, entre otros.

Tipos de Fuentes

Fuentes Primarias: Información original y sin procesar como entrevistas, documentos históricos y artículos científicos.

Fuentes Secundarias: Análisis o resúmenes de fuentes primarias, como libros, enciclopedias e informes de análisis.

Fuentes Terciarias: Compilaciones organizadas de fuentes secundarias y primarias, como bibliografías, catálogos y bases de datos.

Forma de Verificación

Para verificar la confiabilidad de una fuente, se recomienda evaluar los siguientes aspectos:

  • Autoría: ¿Quién la escribió y cuál es su credibilidad?
  • Fecha de publicación: ¿Es actual o ha quedado obsoleta?
  • Propósito: ¿Es objetiva o tiene un sesgo?
  • Referencias verificables: ¿Cita otras fuentes confiables?
  • Revisión por expertos: ¿Ha sido evaluada en un proceso académico o científico?

Dónde Buscar

Algunas plataformas confiables para encontrar fuentes de información son:

  • Google Scholar
  • Scielo
  • Redalyc
  • Bases de datos académicas como IEEE, Springer y Elsevier

Estrategias de Búsqueda

Para encontrar fuentes de información de manera eficiente, se recomienda:

  • Uso de palabras clave: Identificar términos relevantes para la búsqueda.
  • Operadores booleanos: Usar "AND", "OR" y "NOT" para refinar resultados.
  • Fuentes confiables: Priorizar revistas académicas, sitios oficiales y bases de datos reconocidas.
  • Filtros de búsqueda: Utilizar filtros por fecha, idioma y tipo de documento.
