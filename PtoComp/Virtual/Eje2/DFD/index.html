Registro del estudiante

Primero observa el ejemplo Registro de compra

Caso: una compra en una tienda

Una persona prende el dispositivo de registro, digita el producto que desea comprar, el precio y el dinero disponible. El sistema calcula cuánto lleva gastado y pregunta si todavía alcanza el dinero para agregar otro producto. Al final muestra el total y termina la compra.

Inicio / Fin Prender el dispositivo e iniciar la compra.
Entrada / Salida Digitar producto, precio y dinero disponible.
Proceso Calcular el total gastado.
Decisión ¿Alcanza para otro producto?
Entrada / Salida Mostrar total y dinero restante.
Inicio / Fin Finalizar compra.
