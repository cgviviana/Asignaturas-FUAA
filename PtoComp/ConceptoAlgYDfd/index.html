Algoritmos y DFD

Material visual con tonos kawaii y un ejemplo único: preparar café.

¿Qué es un algoritmo?

Un algoritmo es una secuencia ordenada de pasos que permite resolver una tarea. En este material se usa un solo ejemplo para mantener la idea clara: preparar café. Así es más fácil ver cómo una explicación, un diagrama y una simulación pueden representar la misma lógica.

1

Preciso

Cada paso debe indicar qué se hace y en qué orden se hace.

2

Definido

Si se siguen los mismos pasos, se obtiene el mismo resultado.

3

Finito

Debe terminar después de un número limitado de acciones.

Estructura básica del algoritmo

Entrada

Agua, taza y café.

Proceso

Calentar, preguntar si ya hirvió y verter.

Salida

Una taza de café lista.

☕
➡️

Secuencia

Los pasos se realizan uno después del otro, sin cambiar el orden.

❓

Decisión

El algoritmo formula una pregunta y elige un camino según la respuesta.

🗺️

DFD

El diagrama de flujo dibuja la lógica del algoritmo con símbolos y flechas.

DFD del ejemplo del café

El DFD muestra la misma idea del algoritmo, pero en forma visual. Cada figura tiene una función.

INICIO
Terminal

Se usa para indicar el inicio o el final.

CALENTAR
Proceso

Representa una acción, por ejemplo calentar el agua.

¿HIERVE?
Decisión

Representa una pregunta con dos posibles respuestas: sí o no.

SALIDA
Entrada / salida

Sirve para leer datos o mostrar un resultado.

Diagrama simplificado

INICIO
CALENTAR
¿El agua está hirviendo?
VERTER EN LA TAZA
FIN
Si la respuesta a la pregunta es NO, el flujo vuelve a calentar. Si la respuesta es SÍ, el proceso avanza hasta terminar.

Simulador: preparar café

Este simulador sigue exactamente el DFD del café. La idea es observar cómo el algoritmo avanza, cómo toma una decisión y cómo vuelve a un paso anterior cuando la condición todavía no se cumple.
