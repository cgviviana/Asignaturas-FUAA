¿Qué es un algoritmo?

Un algoritmo es una secuencia finita de pasos claros para resolver un problema. Tiene una entrada, un proceso y una salida. Un diagrama de flujo sirve para representar visualmente esa lógica antes o durante la programación.

1

Preciso

Cada paso indica qué hacer y en qué orden hacerlo.

2

Definido

Con la misma entrada, sigue las mismas reglas y produce el mismo resultado.

3

Finito

Debe terminar en una cantidad limitada de pasos.

Simbología básica de diagramas de flujo

Cada figura expresa una idea distinta. El objetivo es que el lector entienda el proceso sin necesidad de revisar código.

INICIO
Terminal
Indica el inicio o el fin del algoritmo.
X = X + 1
Proceso
Describe una operación, cálculo o instrucción.
LEER
Entrada / salida
Se usa para capturar o mostrar información.
¿SI?
Decisión
Evalúa una condición y divide el camino en sí o no.
Idea clave: un algoritmo dice qué pasos hay que hacer y el diagrama de flujo muestra cómo se conectan esos pasos. Ambos representan la misma lógica, pero en formatos distintos.

💻 pseudocodigo.algo
1PROCEDURE BubbleSort(lista)
2  PARA i desde 0 hasta n - 1
3    PARA j desde 0 hasta n - i - 2
4      SI lista[j] > lista[j + 1]
5        // Intercambiar
6        temp = lista[j]
7        lista[j] = lista[j + 1]
8        lista[j + 1] = temp
9      FIN SI
10    FIN PARA
11  FIN PARA
12END PROCEDURE
