¿Qué es un chatbot?

Concepto

Un chatbot es un programa que simula una conversación con el usuario. Puede responder preguntas, guiar procesos o evaluar conocimientos.

Ejemplos

  • Educación: quizzes, tutorías, asistentes
  • Empresas: soporte, ventas, reservas
Registro en Typebot

Pasos

  • Haz clic en Sign up

Opciones de registro

  • Correo electrónico
Espacio de proyectos

Organización

  • Cada tarjeta representa un bot creado
  • Puedes abrir, duplicar o eliminar

Consejo

Usa nombres claros como “Quiz Informática Grupo 3” para identificar tus bots fácilmente.

Entorno de trabajo

Tipos de bloques

  • Burbujas (Text): mensajes del bot
  • Entradas (Inputs): respuestas del usuario
  • Lógica: condiciones y saltos

Conexiones

Las flechas unen los bloques formando un diagrama visual del flujo de conversación.

Mini-proyecto: Quiz

Estructura

  • Saludo inicial: “¿Quieres comenzar el quiz?”
  • Botones de respuesta: Sí / No
  • Pregunta: “¿Cuál es la capital de Colombia?”
  • Opciones: Bogotá / Cali / Medellín
  • Respuesta automática: Correcto ❇️ o Incorrecto ❌
