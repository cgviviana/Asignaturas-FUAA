¿Qué es un chatbot?
Un chatbot es un programa diseñado para simular una conversación con un usuario. Puede responder preguntas, dar información, guiar un proceso o incluso evaluar conocimientos.

🔹 En educación: permite crear actividades interactivas como quizzes, simulaciones de tutoría o asistentes de consulta. 🔹 En empresas: se usan para atención al cliente, soporte técnico, ventas y reservas.

Typebot nos ofrece un entorno sin programar para crear estos bots fácilmente.
Registro en Typebot
Para comenzar a usar Typebot:

1️⃣ Ingresa a https://typebot.io. 2️⃣ Selecciona “Sign up” y elige un método de registro: - Google (rápido y sencillo) - GitHub (para proyectos) - Correo electrónico (tradicional).

Una vez registrado, accederás al panel principal donde podrás crear tus proyectos.
Espacio de proyectos
El panel inicial de Typebot funciona como un tablero de proyectos.

- Cada tarjeta representa un chatbot creado. - Desde aquí puedes abrir, duplicar o eliminar proyectos. - Para iniciar uno nuevo, haz clic en “Crear un Typebot”.

💡 Consejo: usa nombres claros para organizar mejor tus bots.
Entorno de trabajo
El editor de Typebot está formado por bloques visuales que se conectan:

🔹 Burbujas (Text): mensajes del bot. 🔹 Entradas (Inputs): respuestas del usuario (botones, texto, número). 🔹 Lógica: condiciones y control de flujo.

👉 Construyes la conversación como un diagrama de flujo.
Mini-proyecto: Quiz
Ejemplo de aplicación en clase:

1️⃣ Saludo: “¿Quieres comenzar el quiz?”. 2️⃣ Botones: Sí / No. 3️⃣ Pregunta: “¿Cuál es la capital de Colombia?”. 4️⃣ Opciones: Bogotá / Cali / Medellín. 5️⃣ El bot responde con Correcto ❇️ o Incorrecto ❌.

Esto hace que el aprendizaje sea interactivo y divertido.
Publicación del bot
Cuando tu chatbot esté listo:

🔹 Haz clic en Publish. 🔹 Copia el enlace generado. 🔹 Compártelo con tus estudiantes.

En el plan gratuito: - El bot funciona en la web. - Incluye la marca “Made with Typebot”. - Tiene límite de interacciones (suficiente para un taller).
